# 必須要件
1. データ概要（カラム一覧・型・欠損・基本統計量）
2. 分布可視化（数値：ヒスト/箱ひげ、カテゴリ：横棒）
3. 特徴量間の関係（散布図・クロス集計・相関）
4. 外れ値・偏りの指摘
5. date/datetime があれば時系列プロット
6. seaborn/matplotlib を使用
7. コードはすべて実行可能であること
