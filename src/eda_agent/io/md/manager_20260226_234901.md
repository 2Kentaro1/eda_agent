## 🎯 EDA エージェントへの指示
1.  データセット全体の構造を把握するため、各特徴量のデータ型、欠損値の有無と割合、ユニーク数、および基本的な統計量（平均、中央値、標準偏差、最小値、最大値など）を詳細に報告してください。
2.  目的変数の分布をヒストグラムと箱ひげ図で可視化し、その形状、偏り、および潜在的な外れ値の存在について分析してください。
3.  数値型特徴量と目的変数との間の相関関係を、相関係数行列やヒートマップを用いて可視化し、特に相関の強い特徴量を特定してください。
4.  カテゴリ型特徴量について、各カテゴリにおける目的変数の平均値や分布の違いを、箱ひげ図や棒グラフなどを用いて可視化し、目的変数との関係性を分析してください。
5.  各特徴量の分布を可視化し（数値型はヒストグラムや箱ひげ図、カテゴリ型は棒グラフ）、特に顕著な外れ値や異常なパターンがないかを確認し、報告してください。
6.  もし時間に関する特徴量が存在する場合、時間経過に伴う目的変数および主要特徴量のトレンドを可視化し、季節性や周期性などのパターンを分析してください。
