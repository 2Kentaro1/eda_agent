## 🎯 EDA エージェントへの指示
1.  **目的変数（お弁当販売数）の理解:**
    *   お弁当販売数の時系列推移、分布（ヒストグラム、箱ひげ図）、基本的な統計量（平均、中央値、最大値、最小値、標準偏差）を確認し、報告してください。
    *   特に、曜日別、月別、季節別（四半期など）の販売数の傾向を分析し、可視化してください。
2.  **利用可能な特徴量の確認:**
    *   データセットに含まれる全特徴量のリストアップ、データ型、欠損値の有無とその割合、ユニークな値の数を報告してください。
    *   日付情報から「曜日」「祝日」「月」「四半期」といった新たな時間軸特徴量を生成可能か確認し、その生成方法を検討してください。
3.  **主要な外部要因と販売数の関係分析:**
    *   「曜日」とお弁当販売数の関係を分析し、曜日ごとの販売数傾向を可視化してください。
    *   「天気」（気温、降水量、天気種別：晴れ、曇り、雨など）とお弁当販売数の関係を分析し、特に気温と販売数の散布図、天気種別ごとの販売数の分布などを可視化してください。
    *   可能であれば、「曜日 × 天気種別」の組み合わせがお弁当販売数に与える影響を分析し、可視化してください。
4.  **外れ値と異常値の検出:**
    *   お弁当販売数および主要な数値特徴量について、外れ値や異常値がないか箱ひげ図などで確認し、その特徴を報告してください。
