## 🎯 EDA エージェントへの指示

ビジネス目的を達成するために、まずはデータの全体像と主要な傾向を把握する必要があります。以下の項目に重点を置いて初期分析（EDA）を実施してください。

1.  **データ品質の概観と基本統計量の算出**
    *   各カラムの欠損値の有無、データの型、および基本統計量（平均、中央値、標準偏差、最小・最大値）を確認してください。
    *   数値データの分布をヒストグラムで、外れ値の有無をボックスプロットで可視化してください。

2.  **目的変数の特性把握**
    *   プロジェクトの鍵となる目的変数の分布を確認し、偏りや不均衡（クラスバランスなど）がないか分析してください。

3.  **特徴量と目的変数の相関分析**
    *   各特徴量と目的変数の間の相関関係（相関係数や散布図）を特定し、影響度の高い変数を抽出してください。
    *   カテゴリー変数については、各カテゴリごとの目的変数の平均値や中央値の差異を可視化してください。

4.  **変数間の相互作用の確認**
    *   特徴量同士の相関（多重共線性）を確認し、分析に影響を与える可能性のある変数の組み合わせを特定してください。

5.  **セグメント別・時系列分析（データに含まれる場合）**
    *   時間軸や主要な属性（地域、カテゴリ等）による傾向の変化を折れ線グラフなどで確認してください。
