データクリーニングエージェントとして、提供されたDataFrameのクリーニングを行います。

マネージャーからの指示に記載されているEDAの目的（早期離反の分析）と、提供されたDataFrameのデータ内容（レストランの販売データと思われる）に不一致が見られます。しかし、「Your Task」には「上記の DataFrame をクリーニングしてください」と明確に指示されているため、提供されたDataFrameの内容に基づき、一般的なデータクリーニングを実施します。

### クリーニング計画

1.  **データ型の変換**:
    *   `datetime`: `datetime`型に変換します。
    *   `precipitation`: `--`を`0.0`に置き換え、`float`型に変換します。
2.  **欠損値の処理**:
    *   `kcal`: 欠損値（NaN）を中央値で補完します。
    *   `remarks`: 欠損値（NaN）を文字列`'なし'`で補完します。
    *   `event`: 欠損値（NaN）を文字列`'なし'`で補完します。
    *   `payday`: 欠損値（NaN）を`0`で補完します（NaNは給料日ではないことを意味すると仮定）。
3.  **特徴量エンジニアリング（日付関連）**:
    *   `datetime`列から、`year`, `month`, `day`, `day_of_week_name`（曜日名）, `is_weekend`（週末フラグ）を抽出します。

```python

df.info()

print(df.isnull().sum())

# --- 1. データ型の変換 ---
# datetime列をdatetime型に変換
df['datetime'] = pd.to_datetime(df['datetime'])

# precipitation列の'--'を'0.0'に置き換え、float型に変換
df['precipitation'] = df['precipitation'].replace('--', '0.0').astype(float)

# --- 2. 欠損値の処理 ---
# kcal列の欠損値を中央値で補完
# まず中央値を計算 (NaN値を除外して計算)
median_kcal = df['kcal'].median()
df['kcal'].fillna(median_kcal, inplace=True)

# remarks列の欠損値を'なし'で補完
df['remarks'].fillna('なし', inplace=True)

# event列の欠損値を'なし'で補完
df['event'].fillna('なし', inplace=True)

# payday列の欠損値を0で補完 (NaNは給料日ではないと仮定)
df['payday'].fillna(0, inplace=True)
# paydayをint型に変換（0.0 or 1.0から0 or 1へ）
df['payday'] = df['payday'].astype(int)

# --- 3. 特徴量エンジニアリング（日付関連） ---
df['year'] = df['datetime'].dt.year
df['month'] = df['datetime'].dt.month
df['day'] = df['datetime'].dt.day
df['day_of_week_name'] = df['datetime'].dt.day_name(locale='ja_JP.utf8') # 日本語曜日名
df['is_weekend'] = df['datetime'].dt.dayofweek >= 5 # 0=月, 6=日

print("\n\n--- クリーニング後のDataFrame情報 ---")
df.info()
```
