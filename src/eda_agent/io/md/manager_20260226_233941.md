## 🎯 EDA エージェントへの指示
提供されるデータセットに対し、ビジネス目的の理解と分析の方向性定義のために、以下の探索的データ分析を実施してください。

1.  **データセットの概要把握**:
    *   データセットの行数、列数、データ型、メモリ使用量など、全体像を把握してください。
    *   各カラムの名称とその内容（例: 顧客ID、日付、数値データ、カテゴリデータなど）をリストアップし、初期的な理解を深めてください。

2.  **各特徴量の分布と品質評価**:
    *   **数値特徴量**: ヒストグラムや箱ひげ図を用いて分布を確認し、要約統計量（平均、中央値、標準偏差、最小値、最大値など）を算出してください。外れ値の有無とその影響を確認してください。
    *   **カテゴリ特徴量**: ユニーク値のリストとその出現頻度（カウント）を棒グラフなどで可視化し、分布の偏りを確認してください。
    *   **欠損値**: 全てのカラムにおける欠損値の有無、その数、および割合を特定し、報告してください。

3.  **特徴量間の相関分析**:
    *   数値特徴量間の相関係数を算出し、ヒートマップなどで可視化してください。特に相関が強いペアや、潜在的な多重共線性の兆候がないかを確認してください。

4.  **ビジネス目的に資する主要な変数の仮説立て**:
    *   上記の探索結果から、データがどのようなビジネス課題に対応しうるか、またどのようなインサイトが得られそうかを考察してください。
    *   特に、将来的に「目的変数」となりうる候補や、ビジネス課題に強く関連する「主要な特徴量」となりうる候補をいくつか仮説として挙げ、それらの変数が持つビジネス上の意味合いについて考察を加えてください。
