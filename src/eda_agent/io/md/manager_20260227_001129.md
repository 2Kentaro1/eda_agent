## 🎯 EDA エージェントへの指示
1.  **お弁当販売数（目的変数）の基本分析**:
    *   販売数の基本統計量（平均、中央値、最大値、最小値、標準偏差）を算出し、ヒストグラムや箱ひげ図で分布を確認する。
    *   販売数の時系列推移を可視化し、全体的なトレンドやパターン（増加傾向、減少傾向、周期性など）を把握する。

2.  **時間軸での販売傾向分析**:
    *   曜日ごとの平均販売数を算出し、比較棒グラフで可視化して曜日による傾向を特定する。
    *   月ごと、季節ごとの平均販売数を算出し、傾向を可視化する。
    *   祝日や会社の特定イベント（例：社内イベント、長期休暇）が販売数に与える影響を分析する（関連データが存在する場合）。

3.  **外部要因と販売数の関係性分析**:
    *   利用可能な場合は、天候データ（気温、天気、降水量など）とお弁当販売数の相関を分析し、散布図などで可視化する。特定の天候条件（例：猛暑日、雨の日など）が販売数に与える影響を考察する。

4.  **データ品質と異常値の確認**:
    *   各カラムの欠損値の有無とその割合を確認する。
    *   販売数データにおける外れ値を検出し、その特徴を分析する。
