## 🎯 EDA エージェントへの指示

1.  **目的変数（お弁当の販売数）の分布と基本的な統計量の確認:**
    *   お弁当の販売数のヒストグラム、箱ひげ図を作成し、平均、中央値、標準偏差などの統計量を算出してください。
    *   販売数の時系列トレンドを、日別、週別、月別で可視化してください。
2.  **時間的要因と販売数の関係の分析:**
    *   曜日ごとの販売数の傾向を分析し、可視化してください。
    *   祝日や会社の特定日（例：給料日、特定のイベント日など、もしデータにフラグがあれば）が販売数に与える影響を分析してください。
    *   季節性や月ごとの販売数の変動を分析し、可視化してください。
3.  **外部要因（もしデータがあれば）と販売数の関係の探索:**
    *   利用可能な天気データ（例：気温、降水量、天気種別など）がある場合、それぞれの要素と販売数の関係を分析し、散布図や箱ひげ図で可視化してください。
    *   もし社内イベント情報（例：大規模会議、全社イベントなど）がある場合、それらのイベント前後で販売数に有意な変化があるか分析してください。
4.  **データ品質と異常値の確認:**
    *   全データセットにおける欠損値の有無、その割合、および欠損パターンを確認してください。
    *   お弁当の販売数および主要な特徴量（例：気温など）における外れ値を検出・特定し、その影響度について考察してください。
