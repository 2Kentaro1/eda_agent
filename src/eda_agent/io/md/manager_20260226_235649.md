## 🎯 EDA エージェントへの指示

あなたは、ECサイトの売上最大化を目指すための要因を特定するために、以下のEDAタスクを実行してください。

1.  **売上データ（目的変数）の全体像把握:**
    *   月別、週別、日別の売上トレンドを可視化し、季節性や特異日（祝日、キャンペーン期間など）がないか確認してください。
    *   1顧客あたりの平均購入金額、購入回数などの基本的な統計量を算出し、分布を可視化してください。
2.  **顧客属性と売上の関係分析:**
    *   顧客のデモグラフィック情報（年齢層、性別など）、会員ランク、初回購入からの期間といった属性ごとに、平均売上や購入頻度を比較し、売上に貢献している顧客セグメントを特定してください。
3.  **商品属性と売上の関係分析:**
    *   商品カテゴリ、ブランド、価格帯といった商品属性ごとに売上貢献度を分析し、主要な売上ドライバーとなっている商品群や、改善の余地がある商品群を特定してください。
    *   特定の商品やカテゴリの売上推移に特異点がないか確認してください。
4.  **時間的要因と売上の関係分析:**
    *   曜日、時間帯ごとの売上を分析し、購買行動に傾向が見られるか確認してください。
    *   キャンペーン実施期間と非実施期間での売上変化を比較し、キャンペーン効果の有無を評価してください。
5.  **データ品質確認と特徴量間の関係:**
    *   主要な特徴量（顧客属性、商品属性、時間情報など）における欠損値の有無と、その分布を確認してください。
    *   主要な数値特徴量について、分布（ヒストグラム、箱ひげ図など）を可視化し、外れ値の検出を行ってください。
    *   目的変数（売上）と各特徴量との相関関係を分析し、特に相関の強い特徴量を特定してください。
